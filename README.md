# Character Type Checker

A simple yet effective C++ console application that identifies the category of any character entered by the user.

## Features
* *Alphabet Detection:* Recognizes letters and determines if they are Uppercase or Lowercase.
* *Automatic Case Conversion:* If a letter is Uppercase, it shows the Lowercase version (and vice versa).
* *Digit & Symbol Support:* Correct identifies numeric digits and special symbols.

## How to Run

### Using an IDE (Visual Studio / Dev-C++)
1. Open your IDE and create a new C++ project.
2. Copy the code from CharacterChecker.cpp into your main file.
3. Press *F11* (Dev-C++) or *F5* (Visual Studio) to compile and run.

### Using Command Line (GCC)
1. Navigate to the folder: cd path/to/folder
2. Compile: g++ CharacterChecker.cpp -o CharChecker
3. Run: ./CharChecker
